## 二、场景一：拿到VPS后如何第一次连上它

购买完成后，搬瓦工会发一封邮件，包含账户信息。但不管有没有收到邮件，最保险的方式都是直接进KiwiVM后台获取。

### 第一步：进入KiwiVM控制面板

登录搬瓦工官网后台，进入 **Services → My Services**，找到你购买的VPS，点击"KiwiVM Control Panel"按钮进入。

KiwiVM首页就能看到两个关键信息：
- **Public IP address**：你的服务器IP
- **SSH Port**：SSH连接端口（搬瓦工不用默认22端口）

### 第二步：获取root密码

新购买的VPS需要手动重置密码。在KiwiVM左侧菜单找到 **Root password modification**，点击 "Generate and set new root password"，密码会发到你注册邮箱。

> 注意：重置密码需要VPS处于开机状态。

### 第三步：SSH连接

拿到IP、端口、密码这三样东西之后，就可以连接了。

**Windows用户**，推荐Xshell或者直接用系统自带的PowerShell：
bash
ssh root@你的IP地址 -p 你的SSH端口


**macOS/Linux用户**，打开终端直接输入上面同样的命令就行。

连接成功之后，会看到命令提示符变成 `root@bandwagonhost:~#` 这样的样子，恭喜，你已经进入自己的服务器了。

## 三、场景二：学Linux的同学，VPS是最好的练习场

很多人学Linux，在本地虚拟机上折腾，搞坏了还要重头再来。VPS的好处是：KiwiVM后台支持**一键重装系统**，五分钟回到初始状态，怎么折腾都不怕。

### 重装系统教程

进入KiwiVM面板，左侧菜单选 **Install new OS**，可以选择Ubuntu、Debian、CentOS等多种系统。2026年建议选 **Ubuntu 24.04 LTS**，社区生态好，教程资源最多。

点击"Reload"确认后等3-5分钟，系统就重装完毕，新密码会发到邮箱。

### 常用入门命令练习

连上SSH之后，可以练习这些最基础的命令：

bash
# 更新系统软件包
apt update && apt upgrade -y

# 查看系统信息
uname -a
lsb_release -a

# 查看磁盘使用情况
df -h

# 查看内存使用
free -h

# 查看运行中的进程
top


这些命令搞熟了，Linux日常操作就入门了一大半。

如果你在KiwiVM面板里找到了 **Root shell – basic** 选项，那是搬瓦工提供的网页版SSH，不用额外下载工具就能直接在浏览器里敲命令，新手临时用很方便。

## 四、场景三：个人建站教程

用搬瓦工搭建个人网站，最主流的方式是 **宝塔面板 + WordPress**。宝塔面板是一个可视化的服务器管理工具，操作非常友好，不需要手动配置Nginx/Apache。

### 安装宝塔面板

SSH连上搬瓦工之后，执行以下命令（Ubuntu/Debian系统）：

bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh ed8484bec


安装过程大约需要5-10分钟。完成后会显示：
- 宝塔面板地址（格式：`http://你的IP:8888/xxxxx`）
- 默认管理员用户名和密码

记下这些信息，用浏览器访问面板地址，就能看到图形化管理界面。

### 安装LNMP环境 + WordPress

进入宝塔面板后：

1. 点击"推荐安装"，选择LNMP（Linux + Nginx + MySQL + PHP）环境一键安装
2. 安装完成后，在"网站"菜单中添加新站点
3. 填写域名（或IP），创建数据库
4. 到"文件"管理器上传WordPress安装包，或者用面板的WordPress一键安装功能

整个过程如果熟悉的话，1小时之内就能跑起来一个WordPress站点。

## 五、场景四：部署AI应用

这是2025年以来最受关注的搬瓦工使用场景。在自己的VPS上部署Dify、n8n、OpenClaw这类开源AI平台，搭建私有化的AI助手或自动化工作流，数据不过三方，完全掌握在自己手里。

### Dify部署教程（以Docker方式为例）

Dify是目前最火的开源AI应用开发平台，GitHub星标超过130k，支持接入DeepSeek、OpenAI、Claude等大模型。

**最低配置要求**：2核CPU、2GB内存、20GB磁盘——对应搬瓦工CN2 GIA-E套餐的入门配置。

**部署步骤**：

**步骤1：安装Docker**

bash
curl -fsSL https://get.docker.com | bash -s docker
systemctl start docker
systemctl enable docker


**步骤2：安装Docker Compose**

bash
apt install docker-compose-plugin -y


**步骤3：拉取Dify代码**

bash
git clone https://github.com/langgenius/dify.git
cd dify/docker


**步骤4：启动Dify**

bash
cp .env.example .env
docker compose up -d


等待镜像下载和服务启动（约5-10分钟），完成后访问 `http://你的IP` 即可看到Dify的初始化页面。

**步骤5：接入大模型**

在Dify设置页面，选择模型提供商（DeepSeek、OpenAI等），填入API Key，就可以开始搭建知识库问答、AI聊天机器人了。

## 九、几个新手最容易遇到的坑

**坑1：找不到官网**

搬瓦工主域名在国内有时访问困难，官方镜像地址是 `bwh81.net`。网上有大量仿冒网站，看到域名奇怪的一律不要点，更不要在上面输入账号密码。

**坑2：付款被拒**

填写注册信息时，国家/地区要和你的付款方式一致。使用支付宝时选中国即可。开了VPN付款反而可能触发风控，关掉VPN再试。

**坑3：SSH连不上**

先检查：VPS是否开机、IP是否被封。在KiwiVM面板里有一个"Check if my IP is banned"工具，可以检测你的IP有没有被屏蔽。如果被封了，只能换机房（免费迁移）或者付费换IP。

**坑4：优惠码用不了**

截至2026年4月，搬瓦工的历史优惠码（BWHCGLUKKB等）已全部失效。2026年2月短暂出现的NODESEEK2026也已失效。目前暂无可用优惠码，有新码出现时各搬瓦工社群会第一时间通知。

**坑5：买了没用**

这个不是笑话，每个月都有人花了钱买VPS，然后不知道怎么用，最后跑来问"我买完之后能用来做什么"。——这篇文章就是给你们写的。
